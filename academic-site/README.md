# Dr. Ali Assaad — Academic Portfolio Website

A polished, fully responsive **bilingual** personal website for **Dr. Ali Assaad**,
Doctor of Business Administration specializing in Marketing Strategies.

## Highlights

- **Bilingual** — English (default) with an Arabic option. The language toggle in the
  header switches every string and flips the whole layout to right‑to‑left (RTL) for Arabic.
  The chosen language is remembered between visits.
- **Light & dark themes** — follows the visitor's system preference and can be toggled
  manually; the choice is remembered.
- **Single self‑contained file** — `index.html` has all styles, scripts and the profile
  photo embedded. No build step, no external requests, no internet required. Just open it.
- **Content** — every detail from the CV is presented professionally:
  hero with portrait and key statistics, profile, areas of expertise, education and
  professional‑experience timelines, a filterable list of 13 research works (with journal
  quartile Q1–Q3 and status badges), 16 taught courses grouped by degree level,
  20 professional certificates, and contact details.

## Usage

Open `academic-site/index.html` in any modern web browser.

To publish it online, upload the contents of the `academic-site/` folder to any static
host (GitHub Pages, Netlify, Vercel, cPanel, …). `index.html` is the entry point and is
completely standalone.
